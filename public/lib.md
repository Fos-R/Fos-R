# Fos-R library

Fos-R also includes a Rust library that exposes the main parts of the software. Its documentation is [here](doc/fosr_lib/index.html). You can add the stable version of Fos-R to a Rust project with:

`cargo add fosr-lib`

For the experimental version:

`cargo add --git https://gitlab.inria.fr/pirat-public/Fos-R.git fosr-lib`.
